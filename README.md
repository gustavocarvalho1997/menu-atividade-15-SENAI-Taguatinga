# menu-atividade-15-SENAI-Taguatinga

Menu realizado para a atividade-15 do curso de Desenvolvedor Front-end SENAI Taguatinga.

O menu foi feito com base nos conhecimentos passados pelo professor Alex Machado (https://github.com/alexmachadoribeiro), assistindo um video no YouTube (https://www.youtube.com/watch?v=CNiH_hkvcDA) e como material de apoio foi utilizado o https://www.w3schools.com/ .

Realizado apenas com o objetivo de estudar e melhorar na linguagem.
